# NeighborNetwork


# Local Development

## Initial

[Setting up your virutal environment](https://flask.palletsprojects.com/en/1.1.x/installation/#create-an-environment)

### Windows
```
python -m pip install --user virtualenv
virtualenv venv
venv\Scripts\activate
pip install -r requirements.txt
```

###  Linux / Mac
```
python3 -m venv venv
. venv/bin/activate
pip install -r requirements.txt
```

## Running Server
```
venv\Scripts\activate
flask run
```
